# 🧮 Calculator App

A simple and responsive calculator application built using modern web technologies. This project performs basic arithmetic operations with a clean and user-friendly interface.

---

## 📌 Features

- ➕ Addition
- ➖ Subtraction
- ✖️ Multiplication
- ➗ Division
- 🧠 Real-time calculation
- 📱 Responsive design (works on mobile & desktop)
- ⚡ Fast and lightweight

---

## 🛠️ Tech Stack

- HTML5
- CSS3
- JavaScript (Vanilla JS)
